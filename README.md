# 🤖 بوت مكتبة الأوامر الذكية ثنائي اللغة (Smart Prompts Library Bot)

بوت تلغرام احترافي ومطور بلغة بايثون لتصفح ونسخ برومبتات (أوامر) الذكاء الاصطناعي بشكل سلس وتلقائي. يدعم البوت اللغتين العربية والإنجليزية، ويحتوي على نظام اشتراك للمشترقين المحترفين (VIP) محمي بواسطة قاعدة بيانات SQLite3.

---

## 🚀 الميزات الأساسية (Features)
- 🌐 **دعم كامل للغتين (Localization):** واجهات كاملة بالعربية والإنجليزية تتغير بناءً على اختيار المستخدم.
- 📋 **نسخ تلقائي بلمسة واحدة:** الأوامر تظهر بنظام التكويد داخل تلغرام لنسخها بمجرد الضغط عليها.
- ⭐ **نظام مشتركين VIP:** قسم خاص بالأوامر المتقدمة لا يفتح إلا بعد تفعيل الحساب من قبل الآدمن.
- 🗄️ **قاعدة بيانات SQLite3:** حفظ لغة المستخدمين، وحالة الـ VIP بشكل دائم وآمن.
- 📢 **أدوات إدارة كاملة (Admin Panel):** إمكانية تفعيل المشتركين لمح لمح، رؤية الإحصائيات، وإرسال بث جماعي (Broadcast) لكل المستخدمين.
